# Personal Portfolio

A clean, single-page personal portfolio built with vanilla HTML, CSS, and JavaScript. Designed to serve both academic and professional purposes — showcasing research, publications, projects, experience, and skills.

## Features

- **Responsive** — mobile-first design that works on all screen sizes
- **Scroll animations** — AOS (Animate on Scroll) for smooth reveal effects
- **Typing animation** — rotating titles in the hero section
- **3D tilt** — interactive hover effect on project cards
- **Sticky navbar** — with active section highlighting and scroll progress bar
- **Icon-rich** — Font Awesome 6 icons throughout
- **No build step** — pure HTML/CSS/JS, deploy anywhere

## Quick Start

```bash
# Clone the repo
git clone https://github.com/Dayamoy/Dayamoy.github.io.git
cd Dayamoy.github.io

# Serve locally
python3 -m http.server 8000
# Open http://localhost:8000
```

## Deploy to GitHub Pages

1. Create a repo named `<your-username>.github.io`
2. Push your files to the `main` branch
3. Go to **Settings → Pages → Deploy from branch `main`**
4. Your site will be live at `https://<your-username>.github.io`

## Customize

- Edit `index.html` to update content (sections, text, links, images)
- Edit `css/style.css` to change colors, fonts, and layout
- Edit `js/main.js` to modify animations and interactions
- Replace images in `Pictures/` with your own
